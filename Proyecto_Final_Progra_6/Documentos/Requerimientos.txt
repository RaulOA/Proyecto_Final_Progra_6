Redacta un prompt claro y estructurado para una IA o asistente, basado en los siguientes requerimientos para el desarrollo de una aplicación web integral:

**Objetivo:** 
Desarrollar una aplicación web de temática libre, aplicando conocimientos de Angular (Frontend), ASP.NET Core (Backend), Entity Framework (acceso a datos), SQL Server (gestión de base de datos) y Crystal Reports (reportes).

**Tecnologías requeridas:**
- Frontend: Angular 15 o superior
- Backend: ASP.NET Core 6 o superior con REST APIs
- Base de datos: SQL Server
- ORM: Entity Framework Core
- Reportes: Crystal Reports
- Control de versiones: GitHub (obligatorio)

**Requisitos funcionales:**
1. **Autenticación de usuarios:**
   - Registro, inicio y cierre de sesión usando tokens JWT.
2. **Gestión de entidades principales:**
   - Al menos 3 entidades con operaciones CRUD completas y validaciones.
3. **Relaciones en base de datos:**
   - Implementar relaciones uno a muchos y muchos a muchos usando migraciones de EF Core.
4. **Interfaz web:**
   - Diseño responsivo e intuitivo, rutas y formularios reactivos en Angular.
5. **Consumo de APIs:**
   - Operaciones CRUD mediante servicios HTTP e interceptores para JWT.
6. **Reportes:**
   - Mínimo 2 reportes con Crystal Reports, exportables a PDF y con filtros.
7. **Dashboard informativo:**
   - Visualización de KPIs y gráficos usando Chart.js o ngx-charts.
8. **Roles de usuario:**
   - Diferenciación de vistas o acciones según el tipo de usuario.

**Temática:**
La temática es libre, pero debe resolver una necesidad real y específica. Ejemplos:
- Plataforma de gestión académica
- Sistema de reservas y programación de citas
- Portal de empleo interno
- Control de inventario y ventas
- Sistema de gestión de donaciones

**Entregables:**
1. Código fuente completo en un repositorio público (GitHub)
2. Manual técnico (PDF) con:
   - Arquitectura del sistema
   - Diagrama de entidades
   - Explicación de componentes
   - Instrucciones de instalación y ejecución
3. Video demo (máx. 10 minutos) o presentación en clase

El prompt debe mantener la información original, solo mejorar la estructura y claridad para su uso por una IA o asistente.
