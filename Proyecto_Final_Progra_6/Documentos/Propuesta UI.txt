Descripción de Módulos y Pantallas

Módulo 1: Acceso y Seguridad

Pantalla 1: Inicio de Sesión
- Propósito: Permitir la autenticación de los usuarios.
- Requisitos Cubiertos: RF001, RF011 (parcial)
- Componentes:
  - Título (texto o logo)
  - Campo de texto para nombre de usuario
  - Campo de texto tipo password para contraseña
  - Botón "Ingresar"
  - Enlace "¿Olvidó su contraseña?"
  - Área de mensajes para errores

Pantalla 2: Asistente de Recuperación de Contraseña
- Propósito: Guiar al usuario para restablecer su contraseña olvidada.
- Requisitos Cubiertos: RF002
- Componentes:
  - Campo de texto para correo electrónico institucional
  - Botón "Enviar Código"
  - Campo de texto para código de verificación
  - Botón "Aceptar" / "Verificar"
  - Campos de texto tipo password para nueva contraseña y confirmación
  - Botón "Actualizar Contraseña"
  - Área de mensajes

Pantalla 3: Cambiar Contraseña (Panel de Usuario)
- Propósito: Permitir a un usuario autenticado cambiar su contraseña.
- Requisitos Cubiertos: RF003
- Componentes:
  - Campo de texto para nueva contraseña
  - Botón "Restablecer Contraseña"
  - Área de mensajes

Módulo 2: Gestión de Usuarios

Pantalla 4: Gestión de Usuarios
- Propósito: Listar, buscar, filtrar y acceder a funciones de creación, edición y eliminación de usuarios.
- Requisitos Cubiertos: RF005, RF009, RF010
- Componentes:
  - Filtros por tipo de usuario y región (menús desplegables)
  - Botón "Buscar" / "Filtrar"
  - Botón "Agregar Usuario"
  - Botón "Activar Usuario"
  - Tabla de usuarios con columnas: Usuario, Nombre, Rol, Región, Teléfono, Acciones (Editar/Borrar)
  - Área de mensajes

Pantalla 5: Formulario de Usuario (Agregar/Editar/Activar)
- Propósito: Registrar, modificar o reactivar usuarios.
- Requisitos Cubiertos: RF006, RF006.1, RF007, RF008
- Componentes:
  - Título dinámico
  - Campo de búsqueda de funcionario (CNP)
  - Campo de texto solo lectura para nombre
  - Campo de texto para usuario (editable)
  - Campo de texto tipo password para clave/clave temporal
  - Menú desplegable para rol
  - Campos de texto para correo, teléfono, celular
  - Menús desplegables para región y subregión
  - Botones "Agregar"/"Modificar" y "Activar"
  - Menú desplegable de usuarios inactivos (solo en modo activar)
  - Área de mensajes

Módulo 3: Gestión de Productores

Pantalla 6: Gestión de Industriales
- Propósito: Listar, buscar y gestionar productores industriales.
- Requisitos Cubiertos: RF013, RF015, RF016
- Componentes:
  - Filtros por cédula y nombre
  - Botón "Buscar"
  - Botón "Agregar Industrial"
  - Tabla de industriales con columnas: Cédula, Nombre, Teléfono, Representante, Acciones (Editar/Eliminar)

Pantalla 7: Formulario de Industrial (Agregar/Editar)
- Propósito: Registrar o modificar industriales y sus agentes autorizados (maestro-detalle).
- Requisitos Cubiertos: RF012, RF014, RF017, RF017.1, RF018
- Componentes:
  - Sección de datos del industrial: identificador, cédula, nombre, fecha, domicilio legal, representante legal, cédula representante, teléfono, fax, correo, foto, checkboxes de documentación, botón "Agregar"/"Modificar"
  - Sección de agentes autorizados: campos para cédula, nombre, teléfono, correo, estado, región, foto, botón "Agregar Agente (+)", tabla de agentes registrados con acciones

Nota: La gestión de organizaciones sigue la misma estructura, adaptando campos específicos.

Módulo 4: Registros y Procesos

Pantalla 8: Listado de Registros de Productor (Siembras)
- Propósito: Buscar, filtrar y acceder a registros de siembra.
- Requisitos Cubiertos: RF027
- Componentes:
  - Filtros por período, región, cosecha, estado, subregión
  - Botón "Buscar"
  - Tabla de registros con columnas: Consecutivo, Cédula, Productor, Cosecha, Fecha, Acciones (Editar)

Pantalla 9: Formulario de Registro de Productor (Maestro de Siembra)
- Propósito: Visualizar y editar información de un registro de siembra (con pestañas).
- Requisitos Cubiertos: RF028, RF028.1, RF029, RF030, RF031
- Componentes:
  - Sección de datos generales: cédula, nombre, foto, consecutivo, cosecha, fecha, autoriza datos, programa verificación, botón "Aplicar"
  - Pestañas: "Datos de Siembra" (formulario y tabla de siembras), "Organizaciones", "Estadística"

Pantalla 10: Gestión de Compras de Cosecha
- Propósito: Listar y filtrar compras realizadas, con acceso a edición.
- Requisitos Cubiertos: RF032
- Componentes:
  - Filtros por período, región, estado, subregión
  - Botón "Buscar"
  - Tabla de compras con columnas: Tipo, Consecutivo, Fecha, Cultivo, Comprador, Cantidad, Acciones (Editar)

Pantalla 11: Formulario de Compra (Productor Independiente / Organización)
- Propósito: Editar detalles de una compra (diferente para productor u organización).
- Requisitos Cubiertos: RF033, RF034
- Componentes:
  - Sección de datos generales: consecutivo, fecha, comprador, agente, organización, agente organización, productor, cultivo, cosecha, periodo, cantidad, precio unitario, fotos, botón "Aplicar"
  - Sección de detalle por productor (solo tipo organización): selección de productor, cantidad, botón "Agregar al Detalle", tabla de detalle

Módulo 5: Configuración y Reportería

Pantalla 12: Mantenimientos Generales
- Propósito: Administrar catálogos simples (períodos, semillas, etc.).
- Requisitos Cubiertos: RF035, RF036, RF037, RF038
- Componentes:
  - Título
  - Botón "Agregar"
  - Campos de entrada (según catálogo)
  - Botón "Guardar"
  - Tabla de registros con columnas variables y acciones (Modificar/Eliminar)

Pantalla 13: Generador de Reportes
- Propósito: Seleccionar filtros y generar reportes del sistema.
- Requisitos Cubiertos: RR001, RR003, RR004, RR005, RR006, RR007, RR008
- Componentes:
  - Menú desplegable para selección de reporte
  - Filtros dinámicos según reporte (región, subregión, período, cultivo, cosecha)
  - Botón "Generar Reporte"

Pantalla 14: Consulta de Cédula de Productor
- Propósito: Búsqueda rápida de productor por cédula.
- Requisitos Cubiertos: RR002
- Componentes:
  - Campo de texto para cédula
  - Botón "Buscar"
  - Panel de resultados con datos y foto
  - Área de mensajes para "Productor no encontrado"
