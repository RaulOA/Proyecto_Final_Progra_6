Bienvenido a QuickApp

Plantilla de inicio para desarrollo rápido de aplicaciones.
Esta plantilla proporciona una forma ágil y conveniente de crear aplicaciones ASP.NET Core 9.0 / Angular 19.

DEMO EN VIVO 1 - Pro | DEMO EN VIVO 2 - Standard

Tecnologías
  • Páginas de plantilla con Angular 19 y TypeScript
  • Backend API RESTful usando ASP.NET Core 9.0 Web API
  • Base de datos con Entity Framework Core
  • Autenticación/Autorización basada en OIDC/OAuth2 y ASP.NET Core Identity
  • Documentación de API usando Swagger
  • Frontend construido con Angular CLI
  • Tematización con Bootstrap 5
Características
  • Plantilla de inicio simple, no "sobre-ingenierizada", para desarrollo rápido de aplicaciones
  • Estructura completa de backend y frontend lista para construir, con gestión de usuarios, inicio de sesión y roles basada en permisos integrada
  • Código Angular/ASP.NET Core limpio y sencillo para comenzar
  • Capa de acceso a datos construida con los patrones Repository y Unit of Work
  • Base de datos Code First
  • Diseño de API RESTful
  • Servicios de Diálogo y Notificación
  • Página y servicio de configuración integrados
  • Internacionalización integrada
  • Tematización con SASS
  • API de correo lista para usar
  • Manejo de Access y Refresh Tokens con WebStorage (autenticación Bearer) - Sin Cookies
  • APIs CRUD
  • etc.
